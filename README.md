# hydro_weaponrepair

Weapon repair benches for ox_inventory

## Features
- Blacklist weapons
- Multiple locations
- Configurable blip per location
- Configurable parts per location
- Optimized 0.00ms on idle and 0.01ms on use

## Dependencies
- ox_lib - https://github.com/overextended/ox_lib/releases
- ox_inventory - https://github.com/overextended/ox_inventory/releases

## Script Installation
- Install all dependencies
- Install ``hydro_weaponrepair``
- Add ``ensure hydro_weaponrepair`` to your server.cfg
- Configure the parts, locations, and blacklisted weapons
- Restart your server and enjoy repair benches with``hydro_weaponrepair``
